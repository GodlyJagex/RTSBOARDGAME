# RTSBOARDGAME
RTS Board Game

### Turn Structure
- Start of turn events happen
- Make three Actions
- Choose if and how you want to attack an opponent
- End Turn
   
### Buildings
- Bank - At the start of your turn produce 1 Gold
- Lumber Yard -  At the start of your turn produce 1 Wood
- Stone Mine -  At the start of your turn produce 1 Stone
  
### Actions 
- Prodcuce a resource
- Build a building
- Produce a soldier
- Dismantle a building
- Attack an enemy building or townhall
